# LCD Screen Example

## Description

The model of the screen is the 128 x 64 Dot Matrix SSD1306 

## Pins used

The pins used of the discovery board are:

Pin # | Function | Peripheral |
----|----|----|
29 | PB10 | I2C2 SCL (J3) | 
30 | PB11 | I2C2 SDA (J3) | 


The alternate function used is 4.
